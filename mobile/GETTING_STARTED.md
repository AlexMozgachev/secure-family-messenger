# Начало работы с Мобильным Приложением

## Статус: 🚧 В разработке

Мобильное приложение Secure Messenger будет доступно в следующем обновлении.

## Что будет реализовано

### Основные возможности
- ✅ E2E шифрование сообщений
- ✅ Личные и групповые чаты
- ✅ Отправка файлов и медиа
- ✅ Голосовые и видео звонки
- ✅ QR-код онбординг
- ✅ Биометрическая аутентификация

### Технологии
- Expo (React Native)
- WebRTC для звонков
- expo-crypto для шифрования
- SecureStore для безопасного хранения ключей

## Текущая структура проекта

```
mobile/
├── src/
│   ├── screens/         # Экраны приложения
│   ├── components/      # Переиспользуемые компоненты
│   ├── navigation/      # Навигация
│   ├── services/        # API сервисы
│   ├── crypto/          # E2EE криптография
│   └── utils/           # Утилиты
├── assets/              # Ресурсы
└── app.json            # Конфигурация Expo
```
