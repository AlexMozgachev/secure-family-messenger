## 🔄 Как обновиться

### Вариант 1: Чистая установка (рекомендуется)

```bash
# Новая установка v2.0 на чистый Ubuntu 22.04
curl -fsSL https://raw.githubusercontent.com/DenKedr/Secure-Messenger-Builder/main/install_ubuntu22_simple.sh | sudo bash
```

### Вариант 2: Обновление существующей установки

```bash
# Остановить старые сервисы
sudo systemctl stop secure-messenger-*

# Обновить код
cd /opt/Secure-Messenger-Builder
git pull

# Запустить новую установку поверх
sudo bash install_ubuntu22_simple.sh
```

## 🔒 SSL Сертификаты

В v2.0 SSL сертификаты устанавливаются через админ-панель:

1. Зайдите в **Настройки** → **SSL Сертификаты**
2. Нажмите **"Установить SSL сертификат"** 
3. Система автоматически настроит Let's Encrypt
4. SSL будет обновляться автоматически
