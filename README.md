# Семейный защищённый мессенджер

Автономный мессенджер с end-to-end шифрованием, развёрнутый на собственном сервере.

## Особенности
- **E2E шифрование** – сервер не видит содержимое сообщений
- **Real-time** – WebSocket для мгновенной доставки
- **Админ-панель** – управление пользователями, системный мониторинг, блокировка IP, управление SSL
- **Безопасность** – HTTPS (Let's Encrypt), защита от несанкционированного доступа
- **Доступ из интернета** – настройка проброса портов, динамический DNS (DuckDNS)
- **Мобильное приложение** – через Expo (React Native)

## Технологии
- **Backend:** Python, FastAPI, MongoDB, WebSocket, PyJWT, bcrypt, E2EE (X25519, ChaCha20-Poly1305)
- **Frontend:** React, React Router, Axios, Tailwind CSS
- **Инфраструктура:** Ubuntu, Nginx, Docker, Let's Encrypt, DuckDNS

## Развёртывание

Подробная инструкция в [INSTALLATION.md]
Кратко:

1. Запустите MongoDB в Docker
2. Установите зависимости бэкенда и запустите uvicorn
3. Соберите фронтенд (`npm run build`) и скопируйте в `/var/www/messenger`
4. Настройте Nginx (прокси на порт 8001, раздача статики)
5. Настройте проброс портов на роутере (80, 443) на IP вашего сервера
6. Получите SSL-сертификат (через DNS challenge или HTTP)

## Использование
- Администратор входит на `https://ваш-домен/admin` (пароль задаётся при установке)
- Создавайте пользователей, чаты, управляйте системой
- Для семьи можно создать обычных пользователей или дать права администратора (если нужен доступ к админ-панели)
